Trabajo para el cefit

Esta página web corresponde al proyecto final desarrollado en el marco del programa de formación del CEFIT. En este sitio se aplicaron los conocimientos adquiridos a lo largo del curso, incluyendo diseño web, estructura HTML, estilos CSS, integración de JavaScript y uso del framework Django. El objetivo principal de este proyecto es demostrar las habilidades técnicas y creativas adquiridas, ofreciendo una solución funcional, visualmente atractiva y adaptada a diferentes dispositivos.
